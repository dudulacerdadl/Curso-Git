SOBRE


Esse site, assim como esse curso, foi desenvolvido por mim, Eduardo Lacerda, sob a supervisão do professor Ricardo Inacio

Eu sou um estudante de Engenharia de Software na UniFil (Centro Universitário Filadélfia), iniciante em programação.

Faço parte atualmente do projeto NPI – Núcleo de Práticas da UniFil, no qual estou tendo contado e aprendizado sobre novas linguagens como HTML5, CSS3, PHP e JavaScript, para desenvolvimento web.

Faço parte atualmente do PAF-ECF, programa realizado pela faculdade que é responsável pela homologação de sistemas de pagamento via ECF.

Para saber mais, entre em contato: GitHub | Instagram | LinkedIn | E-mail
